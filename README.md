# KOTHIN.COM - Ticket Management System 

KOTHIN.COM is a comprehensive desktop-based Ticket Management System built with C# and Windows Forms (WinForms). It provides a complete solution for managing and booking bus and train tickets, featuring distinct interfaces for administrators and regular users.

## Features

### For Users
* **User Authentication:** Secure login, signup, password recovery, and profile editing.
* **Dashboard:** A central hub to navigate booking options.
* **Transportation Modes:** Choose between Bus and Train travel.
* **Route & Schedule Viewing:** Browse available routes and schedules for buses and trains.
* **Seat Selection:** Interactive seat selection for both buses and trains.
* **Payment Integration:** Dedicated payment forms for processing ticket purchases.

### For Administrators
* **Admin Dashboard:** Centralized control panel for system management.
* **Manage Transport:** Add, edit, or remove buses and trains from the system.
* **Manage Users & Admins:** Oversee user accounts and assign administrative privileges.

## Technology Stack
* **Language:** C#
* **Framework:** .NET Framework / Windows Forms (WinForms)
* **IDE:** Visual Studio
* **Database:** Microsoft SQL Server
